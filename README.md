# BiometricUMG

Sistema de control de asistencia y gestión de usuarios desarrollado en Java con almacenamiento en SQL Server.

## Tecnologías
- Java
- JavaFX
- SQL Server
- ODBC
- Generación de códigos QR

## Funcionalidades
- Registro y control de asistencia
- Gestión de usuarios
- Generación de códigos QR
- Módulo de reportes
- Integración con base de datos relacional
